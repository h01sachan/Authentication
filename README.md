# Authentication
# Installation Guide
1. Clone repository:
```bash
$ https://github.com/h01sachan/Authentication.git
```
2. Install dependencies:

```bash
$ npm i
```
3. Create .env file and write<br>
```
DB_CONNECT = <MONGO_URL> 
ACCESS_TOKEN_KEY = <ACCESS_TOKEN_KEY>
REFRESH_TOKEN_KEY = <REFRESH_TOKEN_KEY>
```
4. Run example:
```bash
npm start
```
